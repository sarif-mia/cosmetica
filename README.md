# Cosmetica - Shopify Theme

**Author:** Sarif Mia  
**Version:** 1.0.0  
**GitHub Repository:** [https://github.com/sarif-mia/cosmetica](https://github.com/sarif-mia/cosmetica)

## Overview

Cosmetica is a modern, responsive Shopify theme designed for beauty, cosmetics, fashion, and general retail stores. The theme has been customized from the original Glozin theme with various enhancements and improvements.

## Features

- **Fully Responsive Design** - Looks great on all devices
- **Entire Slide Clickable** - Slides in the slideshow are fully clickable (not just buttons)
- **Modern UI/UX** - Clean and intuitive user interface
- **Fast Loading** - Optimized for performance
- **SEO Friendly** - Properly structured for search engines
- **Customizable** - Easy to customize through theme settings
- **No License Required** - Removed license validation code

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/sarif-mia/cosmetica.git
   ```

2. Upload the theme files to your Shopify admin panel:
   - Go to Online Store > Themes
   - Click "Upload theme"
   - Select the theme zip file

3. Customize the theme settings in the Shopify admin

## Customizations Made

1. **License Removal** - Removed all license validation code to make the theme free to use
2. **Clickable Slides** - Modified slideshow to make entire slides clickable instead of just buttons
3. **Theme Renaming** - Renamed from "Glozin" to "Cosmetica"
4. **Bug Fixes** - Fixed potential nested anchor tag issues in slides

## Theme Structure

- `assets/` - Contains all CSS, JS, and font files
- `blocks/` - Contains reusable theme blocks
- `config/` - Theme configuration files
- `layout/` - Main theme layout files
- `locales/` - Translation files
- `sections/` - Theme sections
- `snippets/` - Reusable code snippets
- `templates/` - Page templates
